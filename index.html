Hotkeys

All the hotkeys listed below can be adapted or modified.
Scroll to see the full content.


Haste

auto 1000 dontlist | if [$hasted == 0 && $inpz == 0] {say 'utani hur' | wait 1000 }
Only use Hast if you are not in Pz.

Eat Food

auto 60000 dontlist | if [$inpz == 0] { eatfood }
Only eat food outsite Pz.

Fast Hand

auto 1 collectitems 'Expedition Backpack' 8097 3043 3040 9018 6526 3368 3021 5794
Change the name of your Backpack and add the items ID you want.

Reconnect

auto 200 | dontlist | if [$hp == 0] {reconnect | set $died 1 | wait 400 | say '!bless'} | if [$died == 1] {gotolabel temple | set $died '0'}
Reconnect and go to a label 'x' (You must set that label in your cavebot).

Leave cave if attacked

auto 200 | listas 'Pk' | ifplayerattacking {stopattack | settargeting off | gotolabel 'pk' | end}
Go to a label 'x' if a player attack you(You must set that label in your cavebot).

Self Potion

auto 100 if [$hppc <= 70] useoncreature 7642 self auto 100 if [$mppc <= 70] useoncreature 237 self
Go to a label 'x' if a player attack you(You must set that label in your cavebot).

Refil Soft Boots

auto 200 if [$itemcount.6530 == 1] say '!soft'

Change gold

auto 200 | listas 'Crystalcoins - Gold Nugget' | stackitems | if [ $itemcount.3043 >= 100 ] equipbelt 3043 | if [$beltslot.id == 3043 && $beltslot.count == 100] useitem 3043 auto 200 | listas 'Platinum coin- Crystalcoins' | stackitems | if [ $itemcount.3035 >= 100 ] equipbelt 3035 | if [$beltslot.id == 3035 && $beltslot.count == 100] useitem 3035

UE Hunt

auto 100 if [$playersaround.10 == 0 && $monstersaround.1 >= 2] { istargeting say 'Exori Gran' } | if [$playersaround.10 >= 1] { istargeting say 'Exori hur'}

auto 500 if [$monstersaround.7 => 0] {stake 7 5995 | loop}

Cavebot

Some useful cavebots scripts.

NPC Talk Travel

Say hi
Wait 1000
Npcsay Carlin
Wait 1000

NPC Buy Potions

1: Say hi
Wait 1000
Npcsay trade
Wait 1000
2:
set $i '238'
Countitems '$i'
Wait 500
if [$count < 600] {buyitems $i 100 | wait 300 | gotolabel 'buymana'}
gotolabel 'smanadone'

NPC Sell Loot

{ npcsay 'hi' | npcsay 'trade' | wait 1000 | end }
{ wait 1000 | countitems 3392 | wait 1000 }
{ sellitems 3392 $count | end }
{ wait 1000 | countitems 3393 | wait 1000 }
{ sellitems 3393 $count | end } npcsay 'bye'

Check Potions

countitems 7643 | wait 200 | set $uhp $count | countitems 268 | wait 200 | set $mpot $count
if [$uhp =< 200 || $mpot =< 400] {gotolabel refil}
else {gotolabel inicio}

Check Cap

if [$cap =< 2000] {gotolabel 'refil' wait 500}

Check SD

countitems 3155 | wait 200 | set $sdrune $count
if [$sdrune =< 300] {gotolabel refil}
else {gotolabel hunt}

Open BP

closeallwindows
wait 2000
openbpitem
wait 2000
openitemnew 9602 1 00

Move Potions to BP

auto 200 moveitems 268 2868
auto 200 moveitems 7643 2867
