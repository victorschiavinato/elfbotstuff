Cavebot

Some useful cavebots scripts.

NPC Talk Travel


    Say hi
    Wait 1000
    Npcsay 'Carlin'
    Wait 1000

NPC Buy Potions


    1:
    Say hi
    Wait 1000
    Npcsay trade
    Wait 1000
    2:
    set $i '238'
    countitems '$i'
    Wait 500
    if [$count < 600] {buyitems $i 100 | wait 300 | gotolabel 'buymana'}
 
    gotolabel 'smanadone'
Note: Change the ID 238 for your potion ID.

NPC Sell Loot


    { npcsay 'hi' | npcsay 'trade' | wait 1000 | end }
    { wait 1000 | countitems 3392 | wait 1000 }
    { sellitems 3392 $count | end }
    { wait 1000 | countitems 3393 | wait 1000 }
    { sellitems 3393 $count | end } npcsay 'bye'
Note: Add more loots item adding the last 2 lines and changing the ID's.

Check Potions


    countitems 7643 | wait 200 | set $hpot $count | countitems 268 | wait 200 | set $mpot $count

    if [$hpot =< 200 || $mpot =< 400] {gotolabel refil}

    else {gotolabel start}
Note: Change the ID of hpot and mpot.

Check Cap


    if [$cap =< 2000] {gotolabel 'refil' wait 500}

Check Pot & Cap


    countitems 7643 | wait 200 | set $hpot $count | countitems 268 | wait 200 | set $mpot $count
    if [$hpot =< 200 || $mpot =< 400 || cap =<500] {gotolabel refil}

    else {gotolabel start}
Note: Change the ID of hpot and mpot.

Check SD


    countitems 3155 | wait 200 | set $sdrune $count

    if [$sdrune =< 300] {gotolabel refil}

    else {gotolabel hunt}

Open BP


    closeallwindows
    wait 2000

    openbpitem

    wait 2000

    openitemnew 9602 1 00
Note: Change the 9602 ID to your Backpack ID.

Move Potions to BP


    auto 200 moveitems 268 2868

    auto 200 moveitems 7643 2867
Note: Ex: this code will move your mpot(268) to your backpack (2868). Backpack must be open.
